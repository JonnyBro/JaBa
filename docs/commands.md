# Команды  
Список команд JaBa и их описания. JaBa имеет около **120 команд** в **8 категориях**!  

#### Содержимое таблицы  
**Название**: Название команды  
**Описание**: Описание команды  
**Использование**: Использование команды (какие агрумены принимает, [] - обязательно, () - необязательно )  
**Откат**: Время, через которое команду можно будет использовать повторно
